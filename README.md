# MGTV Danmu Cleaner

一个用于芒果 TV 网页端的 Tampermonkey / 油猴脚本，用来处理弹幕中额外显示的角色头像和角色名前缀。

例如原始弹幕可能显示为：

```text
[头像] 陈异: 哇 陈异的车来了❤
```

处理后可以变成：

```text
哇 陈异的车来了❤
```

也可以使用测试模式，将头像和昵称替换成 `111`，方便确认脚本是否生效。

## 功能

- 隐藏芒果 TV 弹幕中的角色头像
- 删除弹幕文本前面的角色名前缀
- 支持类似下面的格式：

```text
陈异: 弹幕内容
陈异：弹幕内容
苗靖: 弹幕内容
```

- 只保留真正的弹幕正文
- 适用于 `www.mgtv.com` 网页端

## 安装 Tampermonkey / 油猴

### Chrome / Edge 浏览器

1. 打开浏览器扩展商店。
2. 搜索 `Tampermonkey`。
3. 安装 Tampermonkey 扩展。
4. 安装完成后，浏览器右上角会出现 Tampermonkey 图标。

## 安装脚本

### 方法一：手动创建脚本

1. 点击浏览器右上角的 Tampermonkey 图标。
2. 选择 `管理面板`。
3. 点击 `+` 或 `添加新脚本`。
4. 删除默认模板代码。
5. 将本项目中的脚本代码完整复制进去。
6. 按 `Ctrl + S` 保存。
7. 打开或刷新芒果 TV 视频页面。

## 使用方法

1. 打开芒果 TV 网页端：

```text
https://www.mgtv.com/
```

2. 进入任意视频播放页。
3. 打开弹幕。
4. 如果脚本生效，原本类似：

```text
[头像] 陈异: 哇 陈异的车来了❤
```

会变成：

```text
哇 陈异的车来了❤
```
